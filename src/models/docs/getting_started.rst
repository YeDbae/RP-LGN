Getting Started
===============

Integrate vbll into your project quickly!


Requirements
------------

* Python version >= 3.7


Installation
------------

The easiest way to install VBLL is with pip:

.. code-block:: console

    $ pip install vbll

You can also install by cloning the GitHub repo:

.. code-block:: console

    $ git clone git@github.com:VectorInstitute/vbll.git
    $ cd vbll
    $ poetry install
